easybuilder

Automatically generate the code of the game by drag and drop controls

Preview

About

EasyBuilder was inspired by GameSalad and wxFormBuilder.

You can drag images to compose the UI scene and create some simply game logic such move, rotate, scene switch... All these operations can be translated to codes: Love2D, libgdx, cocos2d-x. So you can use these to complete your game.

More functions will be added, like GameSalad. Then you can finish mostly part of game by it, you only need to code a little part and other work will generated by EasyBuilder.

How to use

Though Library panel you can add scene and layer, use Media tab to load symbols such as images and fonts.

You can drag Media tab's symbol to Stage panel after choose the scene and layer you want. Then move(mouse left drag) and rotate(mouse right drag) these actors at Stage panel.

After you adjust actors, use Library panel's Behaviors tab to create some actions. Select an actor at Stage panel, then drag actions from Behaviors tab to Backstage panel.

Finally use Settings --> Output to select the code type you want.
